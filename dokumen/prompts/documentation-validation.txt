## Running Python Code

Use `run_shell_command` to execute Python:
- Inline: `run_shell_command({"command": "python3 -c \"print(10000 * 0.5)\""})`
- Script: `run_shell_command({"command": "python3 script.py"})`

For multiline Python code, use heredoc syntax:
```
run_shell_command({"command": "python3 << 'EOF'\nimport json\ndata = {'margin': 10000 * 0.5}\nprint(json.dumps(data))\nEOF"})
```

4. **Produce structured output**

## Handling Large Files (>1000 lines)

For files over 1000 lines, use `spawn_subagent` to process sections in parallel:

1. Count lines: `run_shell_command({"command": "wc -l docs/large-file.md"})`
2. Divide into 500-line sections
3. Spawn subagents to analyze each section
4. Aggregate findings from all subagents

Example:
```
spawn_subagent({
  "subagents": [
    {"file_path": "docs/api.md", "start_line": 1, "end_line": 500, "goal": "Find authentication methods"},
    {"file_path": "docs/api.md", "start_line": 501, "end_line": 1000, "goal": "Find authentication methods"}
  ]
})
```

IMPORTANT: Keep sections to 500 lines or fewer for accurate analysis.
